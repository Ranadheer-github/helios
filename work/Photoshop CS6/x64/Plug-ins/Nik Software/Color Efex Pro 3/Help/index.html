Nik Color Efex Pro 3.0 User Help

 

Color Efex Pro 3.0 is the most advanced set of photographic filters for Adobe® Photoshop®. It installs as a plug-in to Adobe Photoshop, Adobe Photoshop Elements, or other image editing applications that conform to the Adobe Photoshop Plug-In standard.

Color Efex Pro 3.0 features U Point®-powered Control Points, nine new powerful filters, and a new resizable user interface.

Color Efex Pro 3.0 filters are designed to make photographic enhancements not only easier, but better. Based on continuous feedback from professionals in Nik Software’s photographer community, Color Efex Pro 3.0 is a suite of proven controls to help photographers save time, get professional results and de-complicate the art of enhancing their images.
